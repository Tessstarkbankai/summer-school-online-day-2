CSS TYPES USED
==============
• Inline  – background image on <body> and background tint on .contact section  
• Internal – <style> block in <head> for headings, ul, attribute selectors  
• External – style.css for main layout, card glass effect, responsive sizing

SELECTORS USED
==============
• Element selector      : ul, li, p, body  
• ID selector           : #profile-pic  
• Class selector        : .card, .shadow, .bio  
• Group selector        : h1, h2  
• Descendant selector   : .contact a, .contact p  
• Attribute selector    : a[href^="mailto"]  

BONUS
=====
• Hover effect on links (.contact a:hover)  
• Subtle card shadow (.shadow class)
